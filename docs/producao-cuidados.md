# Cuidados para versão pública/comercial

## Imagens

O app foi desenhado para o usuário enviar a própria imagem/print para análise. Evite distribuir uma base pública com imagens oficiais ou imagens copiadas de terceiros sem autorização.

## Dados das cartas

Para uma base própria, use:

- cadastro manual;
- importação CSV/JSON;
- dados enviados pelo próprio usuário;
- fonte autorizada/licenciada.

## Segurança

Antes de vender ou liberar para muitas pessoas, implemente:

- troca de senha;
- recuperação de senha;
- limite de upload;
- limite de análises por usuário;
- política de privacidade;
- termos de uso;
- painel administrativo mais completo;
- logs e backups do banco.

## Escala

Para muitos usuários, considere:

- storage de imagens separado;
- compressão automática de imagens;
- fila para OCR/análises pesadas;
- plano pago de banco/hosting;
- monitoramento de erros.
